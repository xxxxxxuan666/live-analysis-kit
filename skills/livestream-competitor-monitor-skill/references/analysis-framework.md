# Analysis Framework

## Core Questions

1. What is the competitor trying to make viewers do?
2. Which anchor scripts create attention, trust, urgency, or conversion?
3. What are viewers asking or resisting in comments?
4. What visual elements support retention or conversion?
5. Which tactics are repeatable, and which depend on temporary events or streamer personality?
6. What can our team test next?

## Script Analysis

- Hook: how the stream stops scrolling or creates immediate relevance.
- Education: how the anchor explains gameplay, product value, rules, or purchase path.
- Trust: proof points, authority, experience, player examples, social proof.
- Pressure: limited-time benefit, ranking, scarcity, event deadline, fear of missing out.
- Objection handling: price, complexity, authenticity, quality, suitability, risk.
- Conversion: exact CTA, path clarity, repeated instruction, moderator support.

## Comment Analysis

- Quantify themes when possible: high / medium / low frequency.
- Separate true audience pain from curiosity and noise.
- Treat repeated unanswered comments as conversion friction.
- Treat repeated positive comments as potential proof material.
- Watch for mismatch between anchor claims and comment concerns.

## Visual Analysis

- Identify what the viewer sees at decision moments.
- Check whether visual proof supports the script.
- Note overlays or pinned comments that reduce repeated questions.
- Track rhythm: demo, explanation, offer, interaction, proof, CTA.

## Common Opportunity Types

- Script gap: competitor answers a concern we do not address.
- Visual proof gap: competitor demonstrates value more clearly.
- Offer gap: competitor lowers friction with package, bonus, or deadline.
- Comment insight: audience repeatedly asks for missing information.
- Rhythm gap: competitor uses stronger interaction loops or CTA frequency.
- Trust gap: competitor provides social proof or authority cues.
